# SYS.SYSTEMS

A minimalistic living archive showcasing HTML/CSS artworks.

## Series

- **Proto:Angel** - Single artwork exploring angelic forms
- **Holy:3nity** - Trinity series (H301, H302, H303)
- **Pilgrims** - Series of 4 pilgrim artworks

## Features

- Pure HTML/CSS artworks (no frameworks)
- Frosted glass navigation
- Keyboard navigation (arrow keys + ESC)
- Fully responsive design
- Custom BPdots typeface
